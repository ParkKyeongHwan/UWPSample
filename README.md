# UWP (Universal Windows Platform)
- Windows 10, Windows Mobile, XBox One, HoloLens에서 실행되는 범용 응용 프로그램 개발이 가능.
- C++, C#, VB.NET, WinJS and XAML, HTML을 사용해서 개발 가능
- Desktop Bridge 기술을 이용해서 WinForm, WPF 응용프로그램을 마이그레이션 할 수 있음.
